|-------------|-----------|

| Card tap | Ripple effect |
| Download start | Slide in from right |
| Progress update | Smooth interpolation |
| Toast appear | Slide up + fade in |
| Modal open | Scale from 0.9 + fade |
| Tab switch | Cross-fade content |

## Accessibility

- Minimum touch target: 44x44 px
- Color contrast ratio: 4.5:1 minimum
- Support for screen readers (labels)
- Reduce motion option support
- Font scaling support (up to 200%)
